Мебельная группа

Профессиональные лакокрасочные материалы для мебельного производства и реставрации — целлюлозные, полиуретановые и акриловые системы YALAZ

Целлюлозная серия

Быстросохнущие нитроцеллюлозные материалы для профессиональной окраски мебели. Отличная адгезия, высокая твёрдость плёнки, широкая цветовая гамма.

Целлюлозная грунтовка

Cellulosic Primer
Описание
Нитроцеллюлозная грунтовка для мебельных поверхностей. Обеспечивает надёжное сцепление финишного покрытия с основой, выравнивает пористость дерева и MDF. Формирует прочную однородную подложку.
Применение
Дерево, MDF, ДСП, фанера. Наносится распылением. Перед нанесением разбавить целлюлозным растворителем до рабочей вязкости.
8–10 м²/л
Расход
1 / 3 / 18 л
Фасовка

Целлюлозная финишная краска

Cellulosic Topcoat
Описание
Финишная нитроцеллюлозная краска для мебели с высокой укрывистостью. Образует гладкое, равномерное покрытие с отличными декоративными характеристиками. Быстро сохнет, легко полируется.
Применение
Мебельные фасады, корпусная мебель, деревянные изделия. Наносится пневматическим или безвоздушным распылением в 2–3 слоя.
10–12 м²/л
Расход
1 / 3 / 18 л
Фасовка

Применение
Декоративная отделка мебели, акцентные элементы интерьера. Наносится распылением после нанесения целлюлозной грунтовки.

Применение
Мебель из дерева и MDF. Наносится распылением в 2–3 слоя с межслойной шлифовкой. Подходит для конвейерного производства.

Акриловая серия

Водоразбавляемые акриловые материалы с низким содержанием VOC. Экологически безопасны, подходят для использования в жилых помещениях и детских.

Акриловая грунтовка

Acrylic Primer Paint
Описание
Водоразбавляемая акриловая грунтовка для мебельных поверхностей. Отличная адгезия, минимальный запах, быстрое высыхание. Экологически безопасна, сертифицирована для применения в детских помещениях.
Применение
Дерево, MDF, ДСП. Разбавляется водой. Нанесение — кистью, валиком или распылением. Шлифуется после высыхания.
10–12 м²/л
Расход
1 / 2,5 / 10 л
Фасовка
