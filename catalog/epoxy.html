Эпоксидные краски

Эпоксидные грунтовки, финишные покрытия, краски для пола и покрытия без растворителей

← Назад к каталогу

Эпоксидная грунтовка

Описание
Двухкомпонентная эпоксидная грунтовка для металлических и бетонных поверхностей. Превосходная адгезия, химическая стойкость, антикоррозионная защита.
Применение
Грунтование стальных конструкций, резервуаров, трубопроводов, бетонных полов перед нанесением финишного покрытия.

Эпоксидная антикоррозионная краска

Описание
Двухкомпонентная эпоксидная антикоррозионная грунт-краска. Высокая химическая стойкость, устойчивость к воде, солям и нефтепродуктам.
Применение
Защита металлоконструкций в агрессивных средах: химические заводы, нефтеперерабатывающие предприятия, морские сооружения.

Эпоксидный шоппраймер

Описание
Быстросохнущая эпоксидная грунтовка для временной защиты металла. Тонкослойное покрытие, совместимое с большинством финишных систем.
Применение
Временная защита стальных конструкций после пескоструйной обработки на период транспортировки и монтажа.

Эпоксидная межслойная краска

Описание
Двухкомпонентная эпоксидная краска для промежуточных слоёв. Увеличивает общую толщину покрытия и обеспечивает дополнительную защиту.
Применение
Промежуточный слой в многослойных антикоррозионных системах для промышленных объектов.

Эпоксидная финишная краска

Описание
Двухкомпонентная эпоксидная финишная краска. Высокий глянец, отличная химическая стойкость, устойчивость к механическим воздействиям.
Применение
Финишное покрытие промышленных объектов, резервуаров, трубопроводов, металлоконструкций.

Эпоксидная краска Гофре

Описание
Эпоксидная краска с текстурным эффектом гофре (тиснение). Создаёт рельефную поверхность, скрывает дефекты основания.
Применение
Декоративная и защитная окраска металлических поверхностей, оборудования, панелей.

Эпоксидный лак для пола

Описание
Двухкомпонентный эпоксидный лак для бетонных полов. Высокая износостойкость, химическая стойкость, глянцевая поверхность.
Применение
Покрытие бетонных полов в гаражах, складах, производственных помещениях, торговых центрах.

Эпоксидная каменноугольная краска

Описание
Эпоксидная краска с каменноугольной смолой. Превосходная водостойкость и химическая стойкость. Для подземных и подводных конструкций.
Применение
Подземные трубопроводы, сваи, подводные части судов, фундаменты, канализационные системы.

Самовыравнивающаяся без растворителей

Описание
Эпоксидное самовыравнивающееся покрытие без растворителей. Создаёт идеально гладкую, бесшовную поверхность пола. Экологически безопасно.
Применение
Промышленные полы, чистые помещения, пищевые производства, фармацевтические предприятия, паркинги.

Эпоксидная краска без растворителей

Описание
Двухкомпонентная эпоксидная краска без содержания растворителей. 100% сухой остаток, минимальная усадка, экологически безопасна.
Применение
Покрытие полов и стен в закрытых помещениях, где недопустимо использование растворителей.

Прозрачная эпоксидная без растворителей

Описание
Прозрачное эпоксидное покрытие без растворителей. Сохраняет естественный вид основания, обеспечивая защиту и глянцевый блеск.
Применение
Защитное покрытие декоративных бетонных полов, каменных поверхностей, художественных наливных полов.
